# Print the correct Instant unit
## Description
Implement the `subtractFromEpoch()` method to subtract the given days from the `EPOCH` and print the result in the given time zone.

## Examples
**Sample Input 1:**
```console
10
Asia/Yerevan
```

**Sample Output 1:**
```console
1969-12-22T04:00+04:00[Asia/Yerevan]
```
