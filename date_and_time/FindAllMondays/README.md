# Find all mondays

## Description
Write a program that reads a year and the number of a month (1-12) and prints the dates of all Mondays of this month in the correct order (from the first to the last).

## Examples
**Sample Input 1:**
```console
2017 1
```

**Sample Output 1:**
```console
2017-01-02
2017-01-09
2017-01-16
2017-01-23
2017-01-30
```
