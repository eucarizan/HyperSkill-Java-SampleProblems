# Replacing 'a' with 'b'

## Description
Write a program that reads a string and replaces all occurrences of the letter `'a'` with the letter `'b'`.

The program should print out the resulting string.

## Examples
**Sample Input 1:**
```console
aaa
```

**Sample Output 1:**
```console
bbb
```

**Sample Input 2:**
```console
bca
```

**Sample Output 2:**
```console
bcb
```
