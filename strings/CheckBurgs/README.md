# Check burgs

## Description
Write a program that reads the name of a city and checks if the name ends with "**burg**".

Keep in mind, a city can have a short name.

The program should output **true** or **false**.

## Examples
**Sample Input 1:**
```console
Augsburg
```

**Sample Output 1:**
```console
true
```

**Sample Input 2:**
```console
Saint Petersburg
```

**Sample Output 2:**
```console
true
```

**Sample Input 3:**
```console
Tu
```

**Sample Output 3:**
```console
false
```
