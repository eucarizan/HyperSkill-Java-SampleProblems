# Check prefix ignoring the case

## Description
Write a program that checks if a given string starts with the prefix "**J**" ignoring the case.

The program should output **true** or **false**.

## Examples
**Sample Input 1:**
```console
Java
```

**Sample Output 1:**
```console
true
```

**Sample Input 2:**
```console
Kotlin
```

**Sample Output 2:**
```console
false
```
