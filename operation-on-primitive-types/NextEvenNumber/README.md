# Next even number

## Description
Given a natural number, not greater than 10000. Output the even number following this number.

**Hint**: Try to write formula using the modulo operator %.

## Examples
**Sample Input 1:**
```console
7
```

**Sample Output 1:**
```console
8
```

**Sample Input 2:**
```console
8
```

**Sample Output 2:**
```console
10
```
