# ExactlyOneParamIsPositive

## Description
Write a program that reads three integer numbers and prints **true** if exactly ONE number is positive (i.e. > 0).

Otherwise, it should print **false**.

## Examples
**Sample Input 1:**
```console
1 1 1
```

**Sample Output 1:**
```console
false
```

**Sample Input 2:**
```console
1 0 -1
```

**Sample Output 2:**
```console
true
```
