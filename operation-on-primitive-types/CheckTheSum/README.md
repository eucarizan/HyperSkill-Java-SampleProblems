# Check the sum

## Description
Write a program that reads the numbers **a**, **b**, **c** and checks if there's a pair of them that adds up to exactly 20.

The program must output **true** or **false**.

## Examples
**Sample Input 1:**
```console
1 2 3
```

**Sample Output 1:**
```console
false
```

**Sample Input 2:**
```console
4 16 7
```

**Sample Output 2:**
```console
true
```
