# Reversing

## Description
Write a program that reads a three-digit number, calculates the new number by reversing its digits, and outputs a new number.

## Examples
**Sample Input 1:**
```console
320
```

**Sample Output 1:**
```console
23
```

**Sample Input 2:**
```console
976
```

**Sample Output 2:**
```console
679
```
