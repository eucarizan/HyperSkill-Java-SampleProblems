# Exponential expression

## Description
Write a program which reads a double vale **_x_** and evaluates the result of x<sup>3</sup> + x<sup>2</sup> + x + 1

**Output data format**

The double result of the expression

## Examples
**Sample Input 1:**
```console
22.5
```

**Sample Output 1:**
```console
11920.375
```
