# Arithmetic expression

## Description
Write a program that reads an integer value n from the standard input and outputs the result of the following arithmetic expression:

```java
((n + 1) * n + 2) * n + 3
```

## Examples
**Sample Input 1:**
```console
3
```

**Sample Output 1:**
```console
45
```
