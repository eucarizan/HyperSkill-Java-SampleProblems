# The sum of integers from a to b

## Description
Print the sum of all integers from **_a_** to **_b_** including both.

It is guaranteed that **_a < b_** in all test cases.

## Examples
**Sample Input 1:**
```console
5
21
```

**Sample Output 1:**
```console
221
```
