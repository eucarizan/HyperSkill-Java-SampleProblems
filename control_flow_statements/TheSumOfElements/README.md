# The sum of elements

## Description
Find the sum of all elements of a sequence, ending with the number 0.

The number 0 itself is not included into the sequence and serves as a sign of cessation.

## Examples
**Sample Input 1:**
```console
3
6
8
0
```

**Sample Output 1:**
```console
17
```
